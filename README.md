# DotNetTasks

Tasks api written in ASP.NET

This is the API for my [tasks](https://github.com/rrickgauer/tasks) project.

## Resource Map

* `/cancellations`
* `/completetions`
* `/events`
* `/events/{:eventId}/labels`
* `/labels`
* `/password`
* `/recurrences`
* `/user`
* `/email-verifications`
